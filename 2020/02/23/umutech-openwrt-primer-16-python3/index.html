跟 UMU 一起玩 OpenWRT（入门篇16）：Python3

为什么 Python3？

  1. Shell 不适合某些复杂运算，尤其是 OpenWRT 用的 ash。

  2. Lua 缺乏某些 SDK，比如说阿里云 SDK 就没有 Lua 版。

  3. Python2 已经过时。

  4. Node.js 在小型设备上不如 Python3 高效。

  5. C、C++ 之流太难了！Go、Rust 还得编译，麻烦。

  6. Perl、Ruby 已没落。
